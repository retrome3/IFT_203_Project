## Project Description
This is a responsive peer tutoring website developed as the final project for IFT 203. The website includes five pages with interactive elements and works on mobile, tablet, and desktop devices.

## How to Run
1. Download all project files
2. Open `index.html` in any web browser (Chrome, Firefox, Safari, etc.)
3. No server, installation, or dependencies required

## Project Structure
- `index.html` - Home page
- `about.html` - About page
- `services.html` - Services page
- `profile.html` - Profile page
- `contact.html` - Contact page
- `css/style.css` - All styles
- `js/script.js` - All JavaScript

## Technologies Used
- HTML5
- CSS3 (with CSS Variables)
- JavaScript (ES6)
- Font Awesome Icons

## Features Implemented
- Responsive design for all screen sizes
- Dark/Light theme toggle
- Mobile navigation menu
- Form validation
- Interactive booking system
